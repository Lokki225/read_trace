# Resolved Issues

This document catalogs problems that have been solved, preventing AI agents from re-solving the same issues.

## Format

Each resolved issue entry includes:
- **Issue ID**: Unique identifier (ISSUE-001, ISSUE-002, etc.)
- **Title**: Brief description of the problem
- **Description**: Detailed problem statement
- **Root Cause**: Why the problem occurred
- **Solution**: How it was fixed
- **Date Resolved**: When the fix was implemented
- **Files Affected**: Which files were modified
- **Prevention**: How to avoid this issue in the future

---

## Issue Template

```
### ISSUE-XXX: [Title]

**Description**: [Detailed problem statement]

**Root Cause**: [Why it happened]

**Solution**: [How it was fixed]

**Date Resolved**: [YYYY-MM-DD]

**Files Affected**:
- [file path]

**Prevention**: [How to avoid in future]

**Related**: [Links to related issues or decisions]
```
